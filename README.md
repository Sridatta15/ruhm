# RUHM Streamlit App

## Getting Started

1. Install dependencies:
   ```powershell
   C:/Users/HP/AppData/Local/Programs/Python/Python314/python.exe -m pip install -r requirements.txt
   ```

2. Run the Streamlit app:
   ```powershell
   streamlit run app.py
   ```

## Files

- `app.py` - Streamlit demo app that verifies imports and shows charts.
- `requirements.txt` - Required Python packages.

## Notes

- The app uses `streamlit`, `pandas`, `numpy`, and `matplotlib`.
- If you want to use a virtual environment, create one in this folder and activate it before installing.
